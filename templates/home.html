The UCF search web service is a publicly available service that allows queries to made against various UCF data. Currently UCF's directory is the only available data source, allowing you to find people, organizations, departments, and buildings at UCF.

Usage

Results from the web service are returned as JSON encoded data. The service runs over HTTP, so all that needs to be done to make a query and view its results is make an HTTP request to the service URL.

Simple Example

The default search will usually be all you need. Simply supply a search term and retrieve the results.

http://search.smca.ucf.edu/service.php?search=Hitt
